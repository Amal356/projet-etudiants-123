# 🚀 Démarrage Rapide - Création Projet Jira

## 📋 Vue d'Ensemble

Vous devez créer **36 User Stories** réparties sur **4 Sprints** dans Jira.

---

## 🎯 Étapes Rapides

### 1. Créer le Projet
- Nom: `Gestion Étudiants`
- Clé: `PROJ`
- Type: Scrum

### 2. Créer l'Epic
- Nom: `Gestion des Étudiants`

### 3. Créer les 4 Sprints
- Sprint 1: API REST de base
- Sprint 2: Enrichissement
- Sprint 3: Architecture Micro Services
- Sprint 4: Tests et Qualité

### 4. Créer les 36 User Stories
Suivez le guide complet: **[GUIDE_JIRA_COMPLET.md](./GUIDE_JIRA_COMPLET.md)**
